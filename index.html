GitHub

GitHub has long been the place where software developers shared their work. I used it in my classes on software development to show how groups could collaborate. The basic tool is a command called "git" which is rather complex but extremely powerful. The whole thing is based on "repositories", which are directories whose tree has a certain structure, each paired with a remote repository on GitHub. Recently they started running a service called GitHub Pages which makes git available for websites that are to be used for sharing information and projects in general, in particular for personal websites. Once you have obtained username in GitHub you can set up a repository username.github.io, and that will make a website http[s]://username.github.io automatically available. There are then various ways to set up other repositories for projects if you wish.
